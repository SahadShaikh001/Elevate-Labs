# ✅ Console-Based To-Do List App (Python)

### 📋 Description
A simple command-line To-Do List app that lets users:
- Add new tasks
- View all tasks
- Remove completed tasks  
Tasks are saved in a text file (`tasks.txt`) for persistence.

### 🧠 Concepts Used
- Lists
- File Handling (open, read, write)
- Loops & Conditionals
- String Manipulation
